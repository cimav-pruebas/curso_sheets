Open Google Spreadsheet

Open Google Docs Spreadsheet Online

Explanation

To open new spreadsheet in Google Docs we have two options
Option 1:
Step 1: Search Google Docs Spreadsheet in the browser.
Step 2: It will provide the list of search results. Click on the first link (Google Spreadsheet - Google Docs) or directly search using the URL https://docs.google.com/spreadsheet/

Step 3: If the Gmail account is already logged in, it will directly take us to the Google Docs Spreadsheet. If not, log-in with your Gmail account to open the new Google Docs Spreadsheet.
Step 4: The new spreadsheet file will be created as given in the image below,

Option 2:
Step 1: In the browser, search as Google Docs
Step 2: From the list of the search result, click the first link (Google Docs) or directly type using the link https://docs.google.com/

Step 3: Log-in into the Gmail account to open a new Google Docs Spreadsheet. If the Gmail account is logged in then the Google Docs Spreadsheet will be opened automatically.
Step 4: In the Google Drive, click CREATE → Spreadsheet

Step 5: The new Google Docs Spreadsheet will be created as given in the below image.

Layout

Google Docs Spreadsheet Layout

Explanation

Layout is the over all page view of the window.
In Google Docs Spreadsheet, layout is spilt into following formats.
Spreadsheet file name: Given name of the Spreadsheet file will be displayed here.

Sheets home: Sheets home option is present in the left side of the menu bar.

Menu Bar: Menu bar is the horizontal bar that acts as the common housing supply for the window. It is located in the top of the spreadsheet file. It includes the following options like File, Edit, View, Insert, Format, Data, Tools.

Tool Bar: Tool bar is right below the menu bar.
It includes the set of options to modify the file in the user defined format. Standard option in the tool bar are Print, Undo, Redo, Font, Filter, Bold, Italics, Underline and Alignment options.

Formula Bar: Formula Bar has been labelled with the symbol (fx)
It is present below the Tool Bar and mainly used to edit or enter the formula.

Working Area: Working area is the place where the data entered and calculation are done. The rectangular box in the working area is called as Cell.
Selected cell is called as Active cell, where the text or numbers can be entered to do calculation.
Column header of the working area is named with the alphabet A,B,C...
Row header of the working area is named with the numbers 1,2,3...

Create a New Google Spreadsheet

Explanation

To open a new file in the Google Docs Spreadsheet use the following options.
Option 1: In the Menu bar, click File → New → Spreadsheet

Option 2: Type "https://docs.google.com/spreadsheets/" in the address bar. In the Sheets home page, click on the plus icon to Create new spreadsheet file.

New spreadsheet file will be created as given in the image below.

Row in Google Docs Spreadsheet

What is a row in Google Docs Spreadsheet?

Explanation

Cells arranged in the horizontal direction are called as Row.
Rows are identified by numbers from 1, 2, 3 etc., in the row header.
In the below image, Row 4 is selected.

Cell Reference

Cell Reference in Google Docs Spreadsheet

How to refer a cell in Google Docs spreadsheet?

Explanation

Cell Reference helps to identify the cell position. Cell Reference is identified with column letter and row number. It is mainly used for referring a cell. While listing the cell reference, Column letter should always come first followed by the row number.
Example: B6

What is a column in Google Docs Spreadsheet?

Explanation

Cells arranged in vertical direction are called as Column. Columns are identified by the alphabets from A to Z in the column header.
In the below image, Column B is selected.
